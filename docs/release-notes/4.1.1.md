# Fovux 4.1.1 Release Notes

Fovux 4.1.1 was the release-readiness overhaul for the local-first YOLO workbench.

## Highlights

- Added canonical `/runs/{id}/stream` SSE metrics while keeping `/runs/{id}/metrics` as a
  compatibility alias.
- Completed the VS Code Studio command manifest, walkthrough actions, CodeLens provider, run
  decorations, active-run status item, profile quick switching, and Doctor tree UX.
- Completed Dataset Inspector, Training Launcher, Export Wizard, and Annotation Editor behavior
  for the principal-developer workflow.
- Raised backend coverage enforcement to 92% and expanded Studio tests for host and webview
  behavior.
- Integrated release SBOM generation and tightened deterministic CI installs.
- Removed unnecessary runtime image Mesa/OpenGL packages and cleaned actionable Trivy findings.

## Upgrade

```bash
uv tool upgrade fovux-mcp
```

Update Fovux Studio through VS Code Marketplace or Open VSX.
