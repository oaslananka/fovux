# Fovux 4.1.0 Release Notes

Fovux 4.1.0 unified the monorepo release surface, aligned package versions, added Studio
compatibility checks, and hardened the CI/security foundation.

## Highlights

- All package version sources were aligned to `4.1.0` and guarded by `scripts/check_versions.py`.
- Fovux Studio gained compatibility checks against the connected `fovux-mcp` backend.
- The first-run walkthrough, Doctor sidebar, privacy badge, and Language Model Tool registrations
  were added to the VS Code extension.
- MCP Registry and Smithery manifests were added for ecosystem discovery.
- Backend CI expanded with cross-OS tests, fuzzing, path-traversal tests, signed artifacts, and SLSA
  provenance.

## Breaking Changes

None.
