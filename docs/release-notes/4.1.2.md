# Fovux 4.1.2 Release Notes

Fovux 4.1.2 is a patch release that publishes the final verified main-branch hardening after
4.1.1.

## Highlights

- Dataset format auto-detection no longer recursively scans arbitrary filesystem roots for VOC
  XML files.
- GitHub Actions install pinned `pnpm@10.33.0` via npm instead of Corepack activation, avoiding
  Windows runner hangs in the Node 22/24 matrix.
- Dependabot, Trivy, CodeQL, and full CI were revalidated on the release head.

## Upgrade

```bash
uv tool upgrade fovux-mcp
```

Update Fovux Studio through VS Code Marketplace or Open VSX.
