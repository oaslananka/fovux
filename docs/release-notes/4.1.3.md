# Fovux 4.1.3 Release Notes

Fovux 4.1.3 publishes the final verified main-branch supply-chain hardening after the 4.1.2
release. It is a backward-compatible patch release.

## Headline Wins

- **Immutable CI dependencies.** GitHub Actions dependencies are pinned to commit SHAs across the
  workflow set.
- **Pinned container base.** The `fovux-mcp` runtime image now uses a digest-pinned Python base
  image so Trivy scans and release builds refer to the same runtime surface.
- **Clean security queues.** The authoritative repository was rechecked after hardening:
  Dependabot, CodeQL, Trivy, and code scanning open alerts are clean.
- **Scorecard without noise.** Scorecard still publishes advisory results, but no longer uploads
  policy findings as code-scanning SARIF.

## Breaking Changes

None.
