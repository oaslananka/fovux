# Run Doctor

The Fovux doctor checks Python support, package availability, CUDA or MPS status, disk space, HTTP transport health, and active runs. Running it early catches environment issues before a long training or export job starts.

## Steps

1. Run `fovux-mcp doctor` from a terminal, or click the doctor button in this walkthrough.
2. Review the System Health view for warnings and failed checks.
3. Use any available fix actions before starting training.
