# Open the Dashboard

The dashboard shows local runs, live mAP and loss streams, and the current server state in one VS Code webview. It uses server-sent events when available and falls back to polling when the server cannot stream.

## Steps

1. Run `Fovux: Open Dashboard` from the Command Palette.
2. Select up to five runs to overlay their metrics.
3. Use the active-run status bar badge to monitor running jobs while you work elsewhere.
